## 基础环境

- 必须要有团队(ahateam)项目的开发权限

	```
	aha-uniapp document 
	```

- 必须在本地安装了vuepress
	```
	npm install -g vuepress
	```

- 必须在本地有能够运行git脚本的工具如
	```
	git Bash
	```

- 每次修改或者创建文档之后必须要在document文件夹下 使用git脚本工具运行deploy.sh脚本
	```
	npm run deploy
	```

- 完成以上步骤即可进行线上预览文档个的效果
